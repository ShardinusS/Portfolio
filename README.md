## À propos

Site vitrine conçu pour présenter mon parcours, mes compétences et mes projets, sans framework.

---

## Stack

- HTML / CSS / JavaScript vanilla
- Lucide Icons
- Google Fonts (Space Grotesk, JetBrains Mono)
- Formspree (formulaire de contact)
- GitHub Pages (hébergement)

---

## Fonctionnalités

- Design responsive (mobile / desktop)
- Animations au scroll et curseur personnalisé
- Lightbox sur les images de projets
- Formulaire de contact fonctionnel
- Navigation par ancres avec sidebar fixe
